# Trabalho 2 - Engenharia de Dados (MinIO + SQL)

Base inicial do projeto para experimentos com Apache Spark, Delta Lake e integração com armazenamento objeto via MinIO.

## Setup rápido com UV

```bash
uv sync
```

Depois do sync, execute comandos com:

```bash
uv run task server
```

ou:

```bash
uv run python
```
